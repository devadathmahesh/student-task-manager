# Smart Student Task Manager (Java CLI Project)

## 📌 Problem
Students often struggle to manage assignments and deadlines efficiently.

## 💡 Solution
This project is a command-line based task manager built in Java that helps students track, manage, and prioritize tasks.

## 🚀 Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Deadline tracking
- Priority levels

## 🛠️ Technologies Used
- Java
- File Handling
- Collections Framework

## ⚙️ Setup Instructions

1. Clone the repository:
   git clone https://github.com/your-username/student-task-manager

2. Navigate to project folder:
   cd student-task-manager/src

3. Compile:
   javac Main.java

4. Run:
   java Main

## 📂 File Storage
Tasks are stored in:
data/tasks.txt

## 📖 Usage
Follow on-screen menu instructions to manage tasks.
